## 📌 Deskripsi Singkat Tugas

Modul ini berfokus pada penambahan fitur audit terhadap semua system call yang dipanggil oleh proses. Log audit ini kemudian hanya dapat diakses oleh proses `init` (PID 1) menggunakan system call baru `get_audit_log()`. Tujuannya adalah untuk meningkatkan keamanan dan visibilitas aktivitas sistem.

---

## 🛠️ Rincian Implementasi

* Menambahkan struktur `audit_entry` dan array `audit_log[]` di `syscall.c`
* Mengedit fungsi `syscall()` untuk mencatat setiap pemanggilan system call
* Membuat system call baru `get_audit_log()` di `sysproc.c`
* Menambahkan deklarasi dan nomor syscall baru di `defs.h`, `user.h`, `usys.S`, dan `syscall.h`
* Membuat program uji baru `audit.c` untuk membaca dan mencetak log audit
* Mengubah `Makefile` untuk menyertakan `audit.c` ke dalam build

---

## ✅ Uji Fungsionalitas

* `audit`: Menampilkan semua system call yang telah terjadi, jika dijalankan oleh PID 1
* Untuk menguji secara langsung, program `audit` dijalankan sebagai proses `init` dengan mengedit `init.c`

---

## 📷 Hasil Uji

### 📍 Output `audit` sebagai PID 1:

```

\=== Audit Log ===
\[0] PID=1 SYSCALL=5 TICK=12
\[1] PID=1 SYSCALL=6 TICK=13
...

```

### 📍 Output jika dijalankan oleh proses biasa:

```

Access denied or error.

```

---

## ⚠️ Kendala yang Dihadapi

* Lupa membatasi akses hanya untuk PID 1 menyebabkan proses biasa bisa membaca log
* Log audit cepat penuh karena belum ada mekanisme circular buffer
* Perlu memodifikasi `init.c` agar `audit` dapat dijalankan sebagai PID 1
